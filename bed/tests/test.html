Instructions:

 

    Unix and Linux (tar.gz files)

    For UNIX and LINUX downloads, the gzipped tar file contains the executable(s) and other files.  The file name format is bed_arch_os_ws.tar (.gz), where arch is the system architecture, os is the operating system, and ws is the windowing system.  For example, bed_sparc_solaris_x.tar is bed for Sun Sparcs running Solaris under X11.  To extract the files, download the gzipped file, ucompress the file (gunzip filename) and untar the resulting tar file in your home directory (e.g. tar -xvf  bed_sparc_solaris_x.tar). Tar will create the directory  ~/bed which will contain tcl script examples.  It will create the directory ~/icons and place a bunch of usefull xpixmaps in there.  It will create (or use) a ~/bin directory in which will be placed all the executables (bed, bcalc, bexp, etc.).  AREADME.TXT is extracted to your home directory.  You will need to copy the icon files into the appropriate directory(s) for your window manager to be able to use them.
     

    Windows NT and 95

    For Windows downloads, the file downloaded is the executable.  Just save the file in the proper directory.  For Windows 95 use C:/Windows/Desktop/bed.exe and for NT C:/Windows/Profiles/..../Desktop/bed.exe.   That is assuming you have windows intalled in the default location.  If you need the TCL examples and/or README file that comes with the .tar.gz files, download one of those also and use winzip to extract the files you need.  To run bed on Windows, you will also need to install TCL v8.0 which can be downloaded freely. following the link below
